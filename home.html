Welcome to Element our Matrix server hosted by Element Matrix Services, curated by your friends from x24.tools!

Matrix is a protocol for decentralized chat, and Element is a popular client for that protocol.

Element exposes capabilities roughly equivalent to Slack, but with some key differences:

  • Security features such as end-to-end encryption and device verification
  • Join chat rooms on other Matrix servers via federation.
  • Join chat rooms hosted on any external service that is compatible with the Matrix Protocol (think ICQ from long ago)
  • Create and share access more easily to 'spaces' containing collections of 'rooms'

... as well as many other interesting and experimental features to get into. Let's explore it!
